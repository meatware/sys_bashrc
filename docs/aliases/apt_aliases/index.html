01 - ubupdate

>> ubupdate():

about: Ubdate & upgrade ubuntu via apt. Then run apt auto-remove

group: base

example: ubupdate

function ubupdate() {

    sudo apt update
    sudo apt upgrade -y
    sudo apt auto-remove
}
